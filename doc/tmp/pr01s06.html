Organization of this book

This book is divided into the following chapters:

  1. Introduction - What is Together Workflow Editor all about ?

  2. The Graphical User Interface - Explanation of the user interface components, menus and toolbars

  3. Dialogs and Property Panels - How to handle editor dialogs and XPDL property panels

  4. XPDL elements - What they are, how they are related and how to create, edit and view them

  5. XPDL from Scratch - How to create your first XPDL file

  6. Configuration - How to tailor the Editor to your needs

  7. Customization - How to extend the Editor with additional functionality

  8. Extended Attribute Reference - XPDL Extended Attributes used by the Editor itself

  9. Build guide - How to compile, build and package your own version from the source code
